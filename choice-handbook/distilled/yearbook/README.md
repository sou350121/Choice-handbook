# yearbook/（年度复盘：2000–2025，短版）

> 作用：给你一个“每年一页”的参照系，用来复盘与做选择。  
> 写法：只保留 **结论 + 为什么（证据锚点/轻量链接）+ 怎么更新（反证/下一步）**。

## 怎么用（最省力）

1) 先从你人生记忆最深的年份读起（通常是危机年或拐点年）。  
2) 每年只问 3 个问题：  
   - 这一年的“世界默认”变了什么？（利率/秩序/平台/叙事）  
   - 如果当时要做选择，最早的停机点是什么？  
   - 放到今天，你应该坚持的那条原则是什么？  
3) 读完后去看对应的长版：`world_understanding/yearbook/YYYY.md`（有 T0-Analog + roadmap 映射、Endgame、探针）。
4) 如果你想用“成功/失败案例”做类比门禁：看 `world_understanding/yearbook/README.md` 的 **生死透镜（First-Principles）** 与 **成功/失败案例图谱（Case Atlas）**。
