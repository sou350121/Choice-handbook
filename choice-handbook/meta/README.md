# meta/（证据联结与高确定性决策）

这里放的是“资料体系”的骨架：让新材料进入后，能够**验证/反证旧猜想**，并把证据与结论联结起来，逐步升级到更高确定性的决策。

## 你会在这里找到什么

- `hypotheses_registry.md`：**猜想/主张注册表**（带 ID、来源、证据、状态、置信度）
- `decisions_log.md`：**决策日志**（把高确定性建议写成可复盘的条目：置信度 + 证据链 + 退出/重启信号）

## 联结规则（最小但强制）

当出现“新的高可信材料”并且**验证/反证**了旧文档中的猜想时，必须做三件事：

1) 在 `hypotheses_registry.md` 里更新该猜想的 **状态与置信度**，并添加新证据链接  
2) 在“旧文档”里加一条 **证据更新**（链接到新材料/新世界理解）  
3) 如果置信度跨过阈值（例如从 50%→80%），则在 `decisions_log.md` 新增/更新对应的决策条目（必须含退出/重启条件）

> 置信度口径：参考 `frameworks/05_reasoning_toolkit.md` 的 `C2.4 确定性阶梯（20%/50%/80%）`。
