# Frameworks（框架库）

这里放可复用的决策模型与方法。你不需要“记住一切”，只要在合适场景调用合适框架。

## 推荐顺序（MVP）
1. `01_decision_quality.md`：提升“过程质量”（信息/假设/可逆性/复盘）
2. `02_horizon_map.md`：短中长期利益统一视角
3. `03_opportunity_and_option_value.md`：机会成本、期权价值、可逆/不可逆分流
4. `04_end_thinking_tangnuo.md`：尽头思维（唐诺）→ 退出条件/停机点
5. `05_comparative_thinking.md`：比较思考（发展史/对比技术/情景与信号）
6. `06_dimensionality_reduction_anatomy.md`：降维解剖（第一性原理→目标函数→变量/算子→形式化）
7. `07_material_classification.md`：材料分类（采访/论文/文章→可用资产）
