# Frameworks（框架库）

这里放可复用的决策模型与方法。你不需要“记住一切”，只要在合适场景调用合适框架。

## 框架路由（默认顺序）

1) **`01_decision_quality.md`（先把问题写清楚）**：信息质量、关键假设、可逆性、复盘点。  
2) **`02_opportunity_and_option_value.md`（可逆分流）**：可逆→买信息期权；不可逆→先设退出条件再下注。  
3) **`03_end_thinking_tangnuo.md`（最常用：设停机）**：把“尽头”翻译成退出条件/重启条件/停机点。  
4) **`04_horizon_map.md`（跨期取舍）**：职业/迁移/长期项目/资产配置：把短中长期收益与代价摊开。  
5) **`05_reasoning_toolkit.md`（材料去噪）**：材料混杂时：分类→抽变量→证据表→类比→最小验证动作。  
6) **`06_time_travel_endgame_simulator.md`（重武器）**：用户点名要 06 / 要 BTC 式强结论 / 新项目或新 repo 需要 Gate 推演时使用；按固定标题输出（标题不可省）。  

> Cursor 一键入口：默认用 `/inbox` 自动路由；需要直达时再用 `/article-digest`（05）、`/event-reasoning`、`/time-travel-06`（06）。
