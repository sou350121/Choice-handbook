## 32.x 矿业/金属/资源（06 硬化案例）

### 32.1 高品位矿（相对成功） vs 低品位矿（失败模式）
- **Outcome**：高品位矿现金成本低、更能抗周期；低品位矿在价格下行期停产。  
- **T0 环境**：大宗价格波动与能源成本上升。  
- **Killer Assumption**：价格高位可以长期覆盖低品位成本。  
- **Wedge**：能源/运输成本上升。  
- **Flywheel**：成本低 → 现金流稳 → 资本投入↑ → 产能与品位维护↑。  
- **First Tripwire**：现金成本接近现货价、品位下滑。  
- **Catastrophic Bound**：现货价持续低于现金成本。  
- **Death Spiral**：价格↓ → 亏损↑ → 维护下降 → 品位更差 → 亏损更大。  
- **Noise**：短期价格反弹。  
- **最短探针（30d）**：测“现金成本/现货价 + 品位趋势”，超过阈值即判高危。  

### 32.2 分阶段开发（相对成功） vs 大跃进扩产（失败模式）
- **Outcome**：分阶段开发控制风险；大跃进扩产易超支延期。  
- **T0 环境**：项目周期长、审批复杂。  
- **Killer Assumption**：一次性大规模建设能节省成本。  
- **Wedge**：审批/施工延误导致资本开支攀升。  
- **Flywheel**：阶段性交付 → 现金流回补 → 再投资 → 进度更稳。  
- **First Tripwire**：工程延期、预算超支。  
- **Catastrophic Bound**：资本成本上升导致项目不可融资。  
- **Death Spiral**：延期 → 成本↑ → 资金紧 → 再延期。  
- **Noise**：短期产能目标宣示。  
- **最短探针（7d）**：监测“进度偏差 + 资本开支超支率”，超过阈值即判项目失控。  

### 32.3 合规治理（相对成功） vs 社会阻力停工（失败模式）
- **Outcome**：良好社区关系与合规可持续；冲突导致停工与许可证风险。  
- **T0 环境**：环保与原住民权益要求提高。  
- **Killer Assumption**：许可证拿到后可以忽略社区关系。  
- **Wedge**：抗议/诉讼/政治压力。  
- **Flywheel**：合规与信任↑ → 运营稳定 → 现金流稳 → 合规投入↑。  
- **First Tripwire**：社区投诉上升、监管调查增加。  
- **Catastrophic Bound**：许可证暂停或项目被叫停。  
- **Death Spiral**：停工 → 现金流↓ → 社区关系恶化 → 复工更难。  
- **Noise**：单次地方政府表态。  
- **最短探针（48h）**：评估“许可风险清单 + 社区冲突热度”，超过阈值即判停工风险。
