## 17.x 地产/建筑/物业（06 硬化案例）

### 17.1 WeWork（失败） vs IWG/Regus（相对稳健）
- **Outcome**：WeWork 的长租短融与高扩张在冲击期崩溃；更资产轻、可调整的模式更稳。  
- **T0 环境**：低利率、资本充裕、办公需求上行。  
- **Killer Assumption**：高入住率与融资窗口会长期存在。  
- **Wedge**：需求下滑 + 融资收紧。  
- **Flywheel**：入住率↓ → 现金流↓ → 扩张放缓 → 品牌与网络效应下降 → 入住率更↓。  
- **First Tripwire**：入住率低于盈亏平衡、租金折扣扩大、融资延迟。  
- **Catastrophic Bound**：融资窗口关闭且租约固定成本无法缩减。  
- **Death Spiral**：现金流恶化 → 违约风险上升 → 融资更难 → 违约加速。  
- **Noise**：单城短期满租与估值上升。  
- **最短探针（30d）**：画“租约到期墙 + 现金流覆盖”，任何季度覆盖率<阈值即视为高危。  

### 17.2 物流仓储 REIT（相对成功） vs 商场 REIT（承压）
- **Outcome**：物流仓储受电商驱动更稳；商场在客流与租户结构迁移中承压。  
- **T0 环境**：电商渗透率持续提升、实体零售承压。  
- **Killer Assumption**：线下客流会长期稳定。  
- **Wedge**：消费渠道迁移与租户破产。  
- **Flywheel**：空置↑ → 租金↓ → NOI↓ → 维护/改造↓ → 空置更↑。  
- **First Tripwire**：主力店撤出、同店销售下滑、租金续约大幅折让。  
- **Catastrophic Bound**：入住率跌破债务覆盖阈值。  
- **Death Spiral**：空置上升 → NOI 下降 → 债务违约风险 → 资产被迫出售 → 估值进一步下跌。  
- **Noise**：短期节假日客流反弹。  
- **最短探针（7d）**：监测 NOI/债务覆盖率与主力店续约率，低于阈值即判结构失守。  

### 17.3 高周转预售开发商（失败模式） vs 租赁型资产运营（相对稳健）
- **Outcome**：预售依赖在融资收紧时断裂；租赁型资产以稳定现金流更能抗压。  
- **T0 环境**：预售资金驱动扩张，资产负债表快速膨胀。  
- **Killer Assumption**：预售回款与再融资能持续滚动。  
- **Wedge**：政策收紧/销量下滑导致现金流断裂。  
- **Flywheel**：预售下降 → 现金流紧 → 延期交付 → 信任下降 → 预售更差。  
- **First Tripwire**：预售去化率下滑、工程款拖欠、信用利差上升。  
- **Catastrophic Bound**：无法完成交付或触发大规模违约。  
- **Death Spiral**：现金流断 → 停工 → 信任崩 → 预售崩 → 更大现金流断。  
- **Noise**：单个项目短期热销。  
- **最短探针（48h）**：列“交付义务 vs 现金流来源”，若主要依赖未来预售即判高危。
