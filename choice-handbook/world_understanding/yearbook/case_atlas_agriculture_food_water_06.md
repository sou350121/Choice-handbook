## 21.x 农业/食品/水（06 硬化案例）

### 21.1 滴灌体系（相对成功） vs 粗放灌溉（失败模式）
- **Outcome**：滴灌提高水效率与稳定产量；粗放灌溉在干旱期产量崩塌。  
- **T0 环境**：水资源约束上升、极端天气频发。  
- **Killer Assumption**：水资源长期充足，气候波动可忽略。  
- **Wedge**：干旱或水权收紧导致灌溉中断。  
- **Flywheel**：水效率↑ → 产量稳定↑ → 现金流稳 → 再投资↑。  
- **First Tripwire**：地下水位下降、水权配额收紧。  
- **Catastrophic Bound**：灌溉中断导致产量大幅下降。  
- **Death Spiral**：产量↓ → 现金流↓ → 维护投入↓ → 产量更↓。  
- **Noise**：单季好天气带来的高产。  
- **最短探针（30d）**：统计“水权配额 + 井水位趋势”，低于阈值即判产量不可持续。  

### 21.2 多元作物组合（相对成功） vs 单一作物出口（失败模式）
- **Outcome**：多元作物组合更能穿越价格周期；单一作物在价格下行期崩溃。  
- **T0 环境**：全球供需波动、贸易政策不确定。  
- **Killer Assumption**：单一作物价格会长期维持高位。  
- **Wedge**：价格下行或出口受限。  
- **Flywheel**：多元组合 → 收入波动低 → 投入稳定 → 产量稳定。  
- **First Tripwire**：单一作物价格跌破成本、出口关税上升。  
- **Catastrophic Bound**：连续季节价格低于现金成本。  
- **Death Spiral**：价格↓ → 现金流↓ → 减少投入 → 产量↓ → 现金流更↓。  
- **Noise**：单季价格高点。  
- **最短探针（7d）**：做“作物收入占比表”，单一作物>阈值即判集中风险高。  

### 21.3 有库存/对冲的粮商（相对成功） vs 被迫现卖农户（失败模式）
- **Outcome**：有储存与对冲能力可平滑价格波动；被迫现卖在低点锁定亏损。  
- **T0 环境**：收割季集中供给导致价格下压。  
- **Killer Assumption**：现货价格足以覆盖成本，无需储存与对冲。  
- **Wedge**：物流与仓储不足。  
- **Flywheel**：库存能力↑ → 价格选择权↑ → 现金流稳 → 投入可持续。  
- **First Tripwire**：仓储利用率饱和、物流瓶颈上升。  
- **Catastrophic Bound**：无法储存且价格跌破现金成本。  
- **Death Spiral**：被迫抛售 → 现金流紧 → 无法投入储存 → 更被动抛售。  
- **Noise**：短期补贴与收购价承诺。  
- **最短探针（48h）**：测“可储存天数 + 现货价/成本比”，低于阈值即判需紧急减产或融资。
