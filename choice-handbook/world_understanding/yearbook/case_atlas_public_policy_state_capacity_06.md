## 34.x 公共治理/国家能力（06 硬化案例）

### 34.1 稳健财政（相对成功） vs 财政透支（失败模式）
- **Outcome**：稳健财政更能穿越危机；财政透支在冲击时失灵。  
- **T0 环境**：外部冲击频发、救助需求上升。  
- **Killer Assumption**：可无限举债或货币化解决短缺。  
- **Wedge**：融资成本上升或信用受损。  
- **Flywheel**：财政稳 → 信用强 → 融资成本低 → 处置能力更强。  
- **First Tripwire**：债务率快速上升、融资成本飙升。  
- **Catastrophic Bound**：融资窗口关闭或货币崩溃。  
- **Death Spiral**：成本↑ → 缩支/加税 → 经济下滑 → 税基更弱 → 成本更高。  
- **Noise**：短期刺激与口号。  
- **最短探针（30d）**：测“债务率变化 + 融资成本”，超过阈值即判财政失控。  

### 34.2 强执行力（相对成功） vs 纸面政策（失败模式）
- **Outcome**：执行力强的政策能落地；纸面政策无法转化为结果。  
- **T0 环境**：政策目标需要跨部门协作。  
- **Killer Assumption**：出台政策就能产生效果。  
- **Wedge**：执行链条断裂或激励错配。  
- **Flywheel**：执行成功 → 信任提升 → 执行更顺畅。  
- **First Tripwire**：地方执行差异大、监督与问责缺失。  
- **Catastrophic Bound**：执行失败导致社会失序或信任崩塌。  
- **Death Spiral**：执行弱 → 信任下降 → 合作差 → 执行更弱。  
- **Noise**：短期数据美化。  
- **最短探针（7d）**：监测“执行一致性 + 反馈闭环速度”，低于阈值即判执行失效。  

### 34.3 高信任社会（相对成功） vs 低信任社会（失败模式）
- **Outcome**：高信任降低交易成本与治理成本；低信任导致合规与执行成本上升。  
- **T0 环境**：危机与冲突增加社会压力。  
- **Killer Assumption**：社会信任不会快速恶化。  
- **Wedge**：公共事件与不公平感。  
- **Flywheel**：信任↑ → 合作↑ → 成本↓ → 结果更好 → 信任更高。  
- **First Tripwire**：抗议频率上升、合规率下降。  
- **Catastrophic Bound**：社会分裂导致政策无法执行。  
- **Death Spiral**：信任↓ → 合作↓ → 结果差 → 信任更↓。  
- **Noise**：短期民调波动。  
- **最短探针（48h）**：测“合规率 + 事件热度”，超过阈值即判信任断裂。
