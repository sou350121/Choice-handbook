## 14.x 媒体/内容/创作者（06 硬化案例）

### 14.1 Netflix（成功） vs Blockbuster（失败）
- **Outcome**：Netflix 用订阅+流媒体迁移存活；Blockbuster 在模式转移中失速。  
- **T0 环境**：宽带普及、内容窗口缩短、用户时间碎片化。  
- **Killer Assumption**：线下租赁与罚金模式可以长期成立；用户不会快速转向流媒体。  
- **Wedge**：订阅制与个性化推荐改变消费路径。  
- **Flywheel**：订阅规模↑ → 数据与推荐↑ → 内容投入↑ → 留存↑ → 订阅规模↑。  
- **First Tripwire**：线下客流持续下降、流媒体占用时长快速上升。  
- **Catastrophic Bound**：内容授权成本高于订阅 LTV 或关键版权撤出。  
- **Death Spiral**：流失↑ → 内容预算↓ → 内容弱 → 留存更差 → 流失更大。  
- **Noise**：单部爆款带来的短期新增。  
- **最短探针（30d）**：测“内容成本/订阅 LTV”和留存曲线，若 2–3 个月无法回正即视为模式不可持续。  

### 14.2 Spotify（成功） vs Pandora（相对失速）
- **Outcome**：Spotify 以订阅+推荐飞轮扩张；Pandora 在授权成本与规模上受限。  
- **T0 环境**：版权成本高、移动端成为主入口。  
- **Killer Assumption**：仅靠广告+广播式体验即可覆盖版权成本。  
- **Wedge**：按需点播与订阅分层提高 LTV。  
- **Flywheel**：付费用户↑ → 议价能力↑ → 曲库与体验↑ → 付费转化↑。  
- **First Tripwire**：版权费用占比不降、付费转化停滞。  
- **Catastrophic Bound**：版权成本>用户 LTV，或核心曲库失去授权。  
- **Death Spiral**：成本↑ → 利润↓ → 投入↓ → 体验弱 → 付费更少。  
- **Noise**：短期榜单或单曲爆红。  
- **最短探针（7d）**：列“版权成本/每付费用户毛利”，低于阈值即默认高危。  

### 14.3 NYT 订阅（成功） vs BuzzFeed（广告依赖失败/收缩）
- **Outcome**：NYT 以订阅与内容资产穿越平台流量波动；广告依赖在算法变化下收缩。  
- **T0 环境**：平台算法掌握分发入口，广告 CPM 波动大。  
- **Killer Assumption**：平台流量会长期稳定且足以覆盖成本。  
- **Wedge**：直达用户关系与订阅。  
- **Flywheel**：订阅收入↑ → 内容与产品投入↑ → 口碑与留存↑ → 订阅↑。  
- **First Tripwire**：平台流量占比过高、CPM 持续下滑。  
- **Catastrophic Bound**：平台算法调整导致核心流量大幅下滑。  
- **Death Spiral**：流量↓ → 广告收入↓ → 内容削减↓ → 流量更↓。  
- **Noise**：病毒式爆款文章。  
- **最短探针（48h）**：把流量来源拆成 3 类，若平台占比>阈值，立即要求订阅转化试验。  

### 14.4 Disney+（相对成功） vs Quibi（失败）
- **Outcome**：Disney+ 依靠 IP 库与家庭入口快速扩张；Quibi 因内容与场景错配失败。  
- **T0 环境**：流媒体竞争激烈，用户注意力碎片化。  
- **Killer Assumption**：短视频也能以订阅收费且用户愿意迁移。  
- **Wedge**：可复用 IP 与多端分发覆盖。  
- **Flywheel**：IP 吸引力↑ → 订阅↑ → 内容投入↑ → IP 扩展↑。  
- **First Tripwire**：试用转正率低、内容完成率低。  
- **Catastrophic Bound**：留存与付费转化长期低于内容成本阈值。  
- **Death Spiral**：留存差 → 内容预算↓ → 内容弱 → 留存更差。  
- **Noise**：上线首月下载量。  
- **最短探针（7d）**：测“试用转正率+内容完成率”，若连续两周低于阈值即判失败。
