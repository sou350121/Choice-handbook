## 25.x 电信/网络/连接（06 硬化案例）

### 25.1 LTE（成功） vs WiMAX（失败/边缘化）
- **Outcome**：LTE 成为全球主流标准；WiMAX 因生态与漫游支持不足被边缘化。  
- **T0 环境**：3G 容量不足、终端生态迅速扩张。  
- **Killer Assumption**：独立标准也能在缺生态下胜出。  
- **Wedge**：终端 OEM 与运营商联盟选择。  
- **Flywheel**：终端支持↑ → 用户↑ → 运营商投资↑ → 覆盖↑ → 终端更多。  
- **First Tripwire**：主流手机不支持、全球漫游缺失。  
- **Catastrophic Bound**：主流运营商集体选择 LTE 并锁定频谱。  
- **Death Spiral**：终端少 → 用户少 → 投资少 → 覆盖弱 → 终端更少。  
- **Noise**：单次测速/峰值速率宣传。  
- **最短探针（30d）**：统计“主流终端支持率 + 运营商承诺覆盖”，低于阈值即判标准战失败。  

### 25.2 FTTH（成功） vs DSL/铜线升级（失速）
- **Outcome**：光纤入户解决带宽与延迟；铜线升级在高需求下失去竞争力。  
- **T0 环境**：视频、云办公与低延迟应用增长。  
- **Killer Assumption**：铜线继续小幅升级即可满足需求。  
- **Wedge**：峰值与上行需求上升。  
- **Flywheel**：光纤用户↑ → 现金流↑ → 覆盖扩张↑ → 用户更多。  
- **First Tripwire**：高端用户流失、ARPU 压力上升。  
- **Catastrophic Bound**：维护成本>升级成本且市场份额持续下滑。  
- **Death Spiral**：用户流失 → 收入下降 → 投资减少 → 体验更差 → 流失更大。  
- **Noise**：单季新增宽带用户小幅反弹。  
- **最短探针（7d）**：对比“竞对光纤覆盖率 + 本网高端用户流失率”，超过阈值即判升级滞后。  

### 25.3 LEO 卫星（相对成功） vs GEO 传统卫星（失速）
- **Outcome**：低轨以低时延与更高容量进入新市场；高轨在低时延需求下失速。  
- **T0 环境**：发射成本下降、时延敏感场景扩张。  
- **Killer Assumption**：高时延仍可满足主要需求。  
- **Wedge**：时延/容量差异与星座规模。  
- **Flywheel**：星座规模↑ → 覆盖与体验↑ → 用户↑ → 现金流↑ → 星座扩张↑。  
- **First Tripwire**：用户对时延/容量抱怨上升、企业客户流失。  
- **Catastrophic Bound**：无法融资扩张星座或关键监管受限。  
- **Death Spiral**：覆盖差 → 体验差 → 用户流失 → 现金流不足 → 覆盖更差。  
- **Noise**：单次发射成功与媒体热度。  
- **最短探针（30d）**：测“时延差距 + 可用容量/用户”，低于阈值即判高轨模式失速。
