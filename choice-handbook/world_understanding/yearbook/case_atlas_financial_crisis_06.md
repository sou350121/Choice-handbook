## 11.x 金融危机（06 硬化案例）

### 11.1 Lehman Brothers（失败）
- **Outcome**：融资窗关闭→挤兑→破产清算。
- **T0 环境**：高杠杆、短融依赖、抵押品价值高度相关；流动性与信任是硬约束。
- **Killer Assumption**：短期融资（repo/CP）永远可续作。
- **Wedge**：抵押品折价扩大 + 对手方提高保证金。
- **Flywheel**：折价↑→追加保证金↑→被迫卖资产→价格↓→折价更↑。
- **First Tripwire**：
  - repo haircuts 连续上升、融资期限显著缩短；或核心对手方减少交易额度。
- **Catastrophic Bound**：
  - 失去“最后贷款人/信用背书”且无法 48h 内补足高质量抵押品。
- **Death Spiral**：
  - haircuts↑ → 流动性缺口↑ → 抛售资产 → 价格↓ → 评级下调 → 更多保证金 → 挤兑。
- **Noise**：单季盈利、复杂对冲“看起来覆盖了风险”。
- **最短探针（48h）**：把融资结构拆成“到期墙”与“抵押品质量”，问：若 haircuts 提高 X%，你 48h 现金缺口是多少？

### 11.2 Bear Stearns（失败）
- **Outcome**：信任崩溃速度极快，几天内融资消失。
- **T0 环境**：高度依赖短融与市场信心；prime brokerage 客户对安全极敏感。
- **Killer Assumption**：市场会给你时间“慢慢去杠杆”。
- **Wedge**：prime brokerage 客户撤离 + repo 续作失败。
- **Flywheel**：客户撤→流动性变差→更难融资→更多客户撤。
- **First Tripwire**：客户资金外流加速、对手方减少授信、融资期限从周/天变成隔夜。
- **Catastrophic Bound**：隔夜融资成为主要来源且没有可用的高质量抵押品缓冲。
- **Death Spiral**：客户撤离 → 融资恶化 → 资产甩卖 → 价格下跌 → 信心更差。
- **Noise**：管理层安抚、媒体口径“有序处置”。
- **最短探针（7d）**：监控“对手方行为”而非价格：授信额度、保证金条款变化、客户留存。

### 11.3 AIG（失败/被救）
- **Outcome**：衍生品保证金条款触发→流动性危机→被救助。
- **T0 环境**：尾部风险被低估，合同条款让风险在极端情景瞬间显性化。
- **Killer Assumption**：极端情景不会同时发生（相关性不会趋 1）。
- **Wedge**：评级下调触发保证金追加。
- **Flywheel**：评级↓→保证金↑→流动性↓→更难维持信用→评级再↓。
- **First Tripwire**：合同中“评级触发条款”暴露度上升、抵押品要求趋严。
- **Catastrophic Bound**：保证金缺口大到只能依赖外部救助。
- **Death Spiral**：评级下调 → 保证金条款触发 → 流动性枯竭 → 系统性风险扩散。
- **Noise**：会计利润平滑、低波动时期的稳定假象。
- **最短探针（30d）**：穿透条款：把所有“评级触发/保证金触发”列成清单，做一次压力测试。

### 11.4 LTCM（失败）
- **Outcome**：价差扩张 + 流动性消失 + 高杠杆 → 被迫救助/清算。
- **T0 环境**：历史回测有效，但结构性相关性在危机中骤变。
- **Killer Assumption**：价差一定会在你能承受的时间窗口内回归。
- **Wedge**：市场流动性消失，价差“对”但交易不了。
- **Flywheel**：亏损→保证金↑→被迫减仓→价差更恶化→亏损更大。
- **First Tripwire**：相关性上升、流动性指标恶化（bid-ask 扩大/成交量塌缩）。
- **Catastrophic Bound**：被迫减仓触发“踩踏”时，市场深度不足以承接。
- **Death Spiral**：相关性趋 1 → 价差扩张 → 保证金↑ → 减仓 → 踩踏。
- **Noise**：模型夏普高、名人背书。
- **最短探针（48h）**：把“能否退出”当第一指标：在最坏情景下，你需要几天/几倍滑点才能平仓？

### 11.5 Northern Rock（失败）
- **Outcome**：批发融资断裂→挤兑→政府介入。
- **T0 环境**：期限错配严重，存款并不稳定（信心驱动）。
- **Killer Assumption**：批发融资永远可得，且存款不会集中挤兑。
- **Wedge**：同业拆借紧张，批发融资成本飙升。
- **Flywheel**：融资成本↑→盈利恶化→信心↓→存款外流→融资更难。
- **First Tripwire**：批发融资占比上升 + 融资期限缩短。
- **Catastrophic Bound**：出现公众信任风暴（排队取款）且无法快速获得央行背书。
- **Death Spiral**：融资恶化 → 媒体传播 → 挤兑 → 现金枯竭。
- **Noise**：资产质量表面还好、短期利润。
- **最短探针（7d）**：把融资来源按稳定性分层，计算“最坏 7 天”现金缺口。

### 11.6 Money Market（Reserve Primary Fund）（失败）
- **Outcome**：打破 1 美元净值→恐慌赎回→流动性危机。
- **T0 环境**：市场把“现金等价物”当成绝对安全，一旦破功就反身性崩。
- **Killer Assumption**：现金管理产品不会出信用事件。
- **Wedge**：持仓里出现不可快速变现/信用受损资产。
- **Flywheel**：赎回→卖不掉→折价→更多赎回。
- **First Tripwire**：赎回加速 + 资产流动性分层恶化。
- **Catastrophic Bound**：净值跌破心理锚点（1.00）。
- **Death Spiral**：净值破锚 → 赎回 → 折价抛售 → 净值更低。
- **Noise**：短期更高收益率。
- **最短探针（48h）**：检查“现金等价物”的资产穿透：最大持仓、流动性等级、单一发行人集中度。

### 11.7 SVB（失败）
- **Outcome**：加息→久期亏损显性化 + 存款集中→协调挤兑→倒闭。
- **T0 环境**：利率 regime 反转，社交媒体/VC 协调让挤兑加速。
- **Killer Assumption**：存款不会同步撤离（相关性不会趋 1）。
- **Wedge**：未实现亏损被市场关注 + 核心客户群体同质。
- **Flywheel**：传言→撤款→被迫卖债→亏损确认→更恐慌。
- **First Tripwire**：存款集中度极高 + 未实现亏损占资本比重上升。
- **Catastrophic Bound**：出现公开的融资/资本动作失败或被误读，引发协调挤兑。
- **Death Spiral**：信心↓ → 撤款↑ → 卖资产确认亏损 → 资本更薄 → 信心更↓。
- **Noise**：会计口径“没卖就不算亏”。
- **最短探针（7d）**：算“在利率上行 X% 下的资本侵蚀” + “前 10 大存款客户同时撤离的现金缺口”。

### 11.8 欧债：希腊（失败/重组）
- **Outcome**：融资成本锁死→债务重组。
- **T0 环境**：无货币主权，财政与竞争力问题无法靠印钞缓冲。
- **Killer Assumption**：外部会无限兜底且条件温和。
- **Wedge**：利差走阔，融资窗口关闭。
- **Flywheel**：利差↑→财政更紧→经济更差→税基更差→利差更↑。
- **First Tripwire**：利差持续上升 + 政治承诺与执行脱节。
- **Catastrophic Bound**：IMF/欧盟援助条件无法满足导致违约。
- **Death Spiral**：融资恶化 → 紧缩 → 经济收缩 → 税基下滑 → 融资更恶化。
- **Noise**：短期政治口号与一次性救助。
- **最短探针（30d）**：看“财政可持续性”而非情绪：利息支出占比、初级财政余额、增长/通胀假设。

### 11.9 日本“失去的十年”（慢性失败）
- **Outcome**：延迟出清→长期低增长。
- **T0 环境**：资产负债表衰退，坏账不出清导致信用传导失灵。
- **Killer Assumption**：靠拖延与温和政策能自然恢复。
- **Wedge**：僵尸企业占比上升，银行资产质量隐瞒。
- **Flywheel**：不出清→低效率→低增长→更不敢出清。
- **First Tripwire**：僵尸化指标上升、投资回报率长期下滑。
- **Catastrophic Bound**：人口/生产率双杀，政策空间耗尽。
- **Death Spiral**：不出清 → 信用不扩张 → 增长低迷 → 再不出清。
- **Noise**：资产价格短期反弹。
- **最短探针（30d）**：衡量“出清速度”：不良确认/重组效率、资源从低效部门迁移的速度。

### 11.10 系统性重要金融机构（SIFI）“不可处置”（失败模式）
- **Outcome**：结构过复杂→危机处置成本极高→要么救助要么崩。
- **T0 环境**：内部担保链条、跨境监管碎片化。
- **Killer Assumption**：复杂性不会在危机中反噬。
- **Wedge**：关键资产无法估值，市场停止交易。
- **Flywheel**：不确定→不交易→无法估值→更不确定。
- **First Tripwire**：报表不可解释、内部交易/担保链条不透明。
- **Catastrophic Bound**：监管介入但处置工具不足，市场信任归零。
- **Death Spiral**：估值失灵 → 市场冻结 → 融资枯竭 → 系统性外溢。
- **Noise**：并购扩张与“协同效应”。
- **最短探针（7d）**：做一次“可处置性审计”：关键实体清单、内部担保、可售资产、处置路径。

### 11.11 UK LDI（2022，险些系统性失败：保证金螺旋）
- **Outcome**：利率冲击→抵押品追加→被迫抛售国债→利率更上行；最终靠央行干预止血。  
- **T0 环境**：低利率时代形成的杠杆策略，在 regime 反转时被保证金条款“瞬间显性化”。  
- **Killer Assumption**：利率波动不会在短时间内达到触发保证金的幅度。  
- **Wedge**：保证金/抵押品条款（variation margin）触发。  
- **Flywheel**：利率↑→保证金↑→抛售国债→价格↓→利率更↑→保证金更↑。  
- **First Tripwire**：保证金需求飙升 + 被迫卖出“最安全资产”（国债）来补抵押品。  
- **Catastrophic Bound**：核心无风险资产市场（国债）流动性失灵（市场冻结/无法报价）。  
- **Death Spiral**：利率冲击 → 保证金追加 → 抛售国债 → 利率更高 → 更大保证金 → 系统性外溢。  
- **Noise**：长期回测的“稳健”，以及“资产安全所以不会出事”的错觉。  
- **最短探针（48h）**：把所有杠杆头寸的保证金条款列清单，问：若利率一天走 X bp，你需要补多少抵押品？从哪里来？

### 11.12 影子银行/回购链条（失败模式：抵押品再质押）
- **Outcome**：抵押品链条断裂→融资窗瞬间关闭→去杠杆踩踏。  
- **T0 环境**：同一份抵押品被多次再质押（rehypothecation），使系统对“单点折价”极敏感。  
- **Killer Assumption**：抵押品永远能被接受、折价变化是渐进的。  
- **Wedge**：抵押品信用/估值被质疑（哪怕只是“暂时”）。  
- **Flywheel**：折价↑→融资额度↓→被迫卖资产→价格↓→折价更↑。  
- **First Tripwire**：haircut 上调、抵押品合格范围收紧、回购期限显著缩短。  
- **Catastrophic Bound**：核心抵押品“失去可接受性”（no-bid/冻结），回购市场失灵。  
- **Death Spiral**：抵押品质疑 → haircuts↑ → 融资枯竭 → 抛售 → 价格崩 → 更大质疑。  
- **Noise**：表面资产质量“还行”、以及“只是技术性流动性问题”的安抚。  
- **最短探针（7d）**：画抵押品链条：这份抵押品被用过几次？谁是最终持有人？若折价上调 5%，现金缺口多大？

### 11.13 货币/汇率防线（1997 亚洲金融危机：短债+外汇储备）
- **Outcome**：短期外债+固定/半固定汇率在冲击下崩，触发资本外逃与深度衰退。  
- **T0 环境**：外币负债、短债占比高；外汇储备与政策信誉是硬约束。  
- **Killer Assumption**：汇率承诺可以在资本外逃下维持。  
- **Wedge**：外汇储备消耗 + 市场开始“测试防线”。  
- **Flywheel**：贬值预期↑→外逃↑→储备↓→更强贬值预期。  
- **First Tripwire**：短债/储备比恶化、外汇远期点数异常、资本流出加速。  
- **Catastrophic Bound**：储备不足以覆盖短债与经常账户缺口，防线失守。  
- **Death Spiral**：测试防线 → 资本外逃 → 储备下降 → 被迫加息/紧缩 → 经济下行 → 信心更差 → 外逃更大。  
- **Noise**：短期高增长与资产价格繁荣。  
- **最短探针（30d）**：计算“短债/储备/经常账户”三件套，并设阈值：一旦短债/储备>1（示意）就进入预警。

### 11.14 2020 “Dash for Cash”（流动性踩踏：安全资产也被抛售）
- **Outcome**：恐慌期资金抢现金，连国债/高等级资产也被抛售；最终依赖央行扩表与市场功能修复止血。  
- **T0 环境**：高杠杆/基差交易/风险平价等策略叠加，平时稳健，冲击时共同去杠杆。  
- **Killer Assumption**：国债/高等级资产永远“可卖、可融、可对冲”。  
- **Wedge**：融资条件变差（haircut 上调/融资期限缩短）+ 波动上升触发风险预算收缩。  
- **Flywheel**：波动↑→风险预算↓→减仓→价格冲击→波动更↑。  
- **First Tripwire**：国债 bid-ask 扩大、回购市场压力、基差扩大且无法套利收敛。  
- **Catastrophic Bound**：核心市场做市功能失灵（no-bid/报价断档），导致“价格发现”停摆。  
- **Death Spiral**：冲击 → 杠杆策略去仓 → 卖出安全资产补保证金 → 市场功能恶化 → 更大去仓。  
- **Noise**：把它当成“基本面坏消息”而忽略融资与市场微结构。  
- **最短探针（48h）**：对你的组合/机构做“融资-波动压力测试”：波动翻倍 + haircuts 上调 X% 时，哪些仓位会被迫卖出？

### 11.15 Archegos（失败：隐形杠杆 × 对手方竞速平仓）
- **Outcome**：单一高度集中的杠杆头寸爆仓，prime broker 竞速减仓导致踩踏与巨额损失外溢。  
- **T0 环境**：总回报互换/衍生品让杠杆与集中度“不在表内”，对手方之间信息不透明。  
- **Killer Assumption**：对手方会给你时间慢慢减仓（且不会同时行动）。  
- **Wedge**：保证金追加触发，且多家对手方同时收紧。  
- **Flywheel**：保证金↑→被迫卖出→价格↓→更多保证金↑→更大卖出。  
- **First Tripwire**：集中度极高 + 对手方数量增加但信息不共享 + 保证金条款趋严。  
- **Catastrophic Bound**：对手方开始“先跑”竞速平仓（race to the exit）。  
- **Death Spiral**：追加保证金 → 强平竞速 → 价格崩 → 更强平 → 更大外溢。  
- **Noise**：表内风险指标看起来很低（因为风险藏在衍生品与对手方结构里）。  
- **最短探针（7d）**：做一次“对手方穿透审计”：每个策略的真实杠杆、集中度、对手方数量、条款触发阈值；并假设“对手方同时收紧”。  

### 11.16 LME Nickel 2022（结构性失败：保证金螺旋 × 市场规则干预）
- **Outcome**：极端波动触发保证金螺旋与流动性枯竭，交易所介入（暂停/取消交易）以阻止系统性崩溃，但引入信任与规则风险。  
- **T0 环境**：仓位集中、可交割供给与挤仓风险、保证金机制在极端情景下放大冲击。  
- **Killer Assumption**：市场规则在极端波动下仍“按既定方式执行”（规则不变）。  
- **Wedge**：保证金上调与强平触发。  
- **Flywheel**：价格跳动→保证金↑→强平→价格更跳→保证金更↑。  
- **First Tripwire**：价格单日极端波动、保证金快速上调、市场深度塌缩。  
- **Catastrophic Bound**：交易所/清算所无法维持有序清算（被迫暂停/回滚），导致规则不确定性上升。  
- **Death Spiral**：波动→保证金→强平→更波动→交易中断→信任下降→流动性更差。  
- **Noise**：把它当成“某个商品的偶发事件”而忽略“保证金螺旋+规则风险”的通用机制。  
- **最短探针（30d）**：对任何“保证金驱动”的市场敞口，强制写：最坏情景下保证金需求、流动性来源、以及“规则被临时改变”的应对预案。

### 11.17 中国地产（Evergrande：失败/出清拖延）
- **Outcome**：高杠杆扩张+预售模型在融资收紧中断裂，债务重组/违约连锁，行业进入长期出清。  
- **T0 环境**：地产依赖滚动融资与预售现金流；政策与信用周期反转会把“现金流资产”变成“资金黑洞”。  
- **Killer Assumption**：融资窗口永远开着，且预售能持续为负债端输血。  
- **Wedge**：融资收紧（政策/银行/信托）+ 预售/信心下降。  
- **Flywheel**：信心↓→预售↓→现金流↓→停工/延期↑→信心更↓。  
- **First Tripwire**：预售下滑、项目停工、信用利差走阔、非标融资断裂。  
- **Catastrophic Bound**：无法完成交付（“烂尾”成为社会事件）导致信心与政策空间同时崩。  
- **Death Spiral**：融资收紧 → 预售下滑 → 交付延迟/停工 → 社会信任崩 → 预售更差 → 更难融资。  
- **Noise**：账面土地储备/资产规模、“国家会救”的想象。  
- **最短探针（30d）**：把“负债滚动能力”拆成三条线：再融资到期墙/预售现金流/政策边界；任一线断就按出清情景估算损失上界。

### 11.18 ABCP/SIV（2007：影子银行短融冻结）
- **Outcome**：资产端长期/复杂，负债端超短；当短融冻结时被迫甩卖资产→价格塌陷→更大冻结。  
- **T0 环境**：资产支持票据与 SIV 依赖市场滚动融资；透明度与估值在压力下迅速失灵。  
- **Killer Assumption**：短融市场永远愿意续作（roll）。  
- **Wedge**：底层资产被质疑（信用/估值）导致 roll 失败。  
- **Flywheel**：roll 失败→被迫卖资产→价格↓→更多质疑→更难 roll。  
- **First Tripwire**：ABCP 发行失败、期限骤缩、利差飙升、评级/担保条款收紧。  
- **Catastrophic Bound**：市场进入“no-bid”，只能折价抛售（价格发现失灵）。  
- **Death Spiral**：质疑→roll 失败→抛售→价格崩→更多质疑→更大冻结。  
- **Noise**：评级与结构化“分层”带来的安全幻觉。  
- **最短探针（7d）**：对任何“短负债长资产”结构，强制画出：到期墙、可抵押资产质量、以及 roll 失败时的应急流动性来源。

### 11.19 养老金/保险“隐形久期”（失败模式：利率 regime 反转）
- **Outcome**：低利率时代堆出的久期与保证收益承诺，在加息周期里变成资本侵蚀与再配置痛苦。  
- **T0 环境**：负债久期长、资产收益不足；通过杠杆/衍生品“补收益”会把风险推迟到 regime 反转。  
- **Killer Assumption**：利率会长期维持低波动或持续下行。  
- **Wedge**：利率快速上行 + 抵押品条款触发。  
- **Flywheel**：利率↑→资产亏损/保证金↑→抛售→利率更↑。  
- **First Tripwire**：久期缺口扩大、对冲不足、保证金需求上升。  
- **Catastrophic Bound**：被迫卖出核心资产以补抵押品，导致“安全资产也不安全”。  
- **Death Spiral**：加息→资产亏损/保证金→抛售→市场功能恶化→更大保证金。  
- **Noise**：只看“长期持有”口径，不看现金流与保证金。  
- **最短探针（48h）**：列出所有“负债承诺×对冲工具×保证金条款”，做一次 1–3 日冲击压力测试。

### 11.20 Terra/UST（失败：稳定币脱锚 × 反身性挤兑）
- **Outcome**：算法稳定币在信心崩塌时进入死亡螺旋，脱锚→赎回/抛售→更脱锚→归零。  
- **T0 环境**：稳定性依赖市场流动性与信心；高收益激励制造“脆弱的需求”。  
- **Killer Assumption**：流动性与信心足以支撑任何规模的赎回冲击。  
- **Wedge**：脱锚触发套利与恐慌撤出。  
- **Flywheel**：脱锚→抛售/铸币→价格更跌→更脱锚→更抛售。  
- **First Tripwire**：脱锚持续、储备/对冲不透明、收益率补贴驱动的 TVL 快速变化。  
- **Catastrophic Bound**：赎回规模超过可用流动性，进入不可逆信心崩塌。  
- **Death Spiral**：脱锚 → 恐慌赎回 → 抛售/铸币稀释 → 价格更低 → 更恐慌。  
- **Noise**：短期 TVL/收益率与“社区信仰”。  
- **最短探针（7d）**：对任何“稳定承诺”，强制回答：在 24h/72h/7d 的赎回冲击下，真实可变现储备是什么？赎回路径是否可被暂停/闸门化？

### 11.21 USDC（2023：储备集中风险 → 脱锚）
- **Outcome**：银行风险外溢到稳定币；当储备集中在受冲击银行，市场先脱锚再等信息澄清。  
- **T0 环境**：稳定币表面是“现金等价物”，但本质是对手方与清算系统风险。  
- **Killer Assumption**：储备分散且随时可兑；银行系统不会在周末/信息真空期失灵。  
- **Wedge**：储备对手方被质疑（银行被接管/限制）。  
- **Flywheel**：信息不确定→脱锚→赎回拥堵→更不确定→更脱锚。  
- **First Tripwire**：储备集中度高、信息披露延迟、赎回通道在非工作时段受限。  
- **Catastrophic Bound**：赎回渠道冻结或储备被永久性减记。  
- **Death Spiral**：对手方风险暴露 → 脱锚 → 赎回挤兑 → 流动性紧张 → 更大脱锚。  
- **Noise**：平时的“1:1 口号”与缺乏压力情景披露。  
- **最短探针（48h）**：把储备穿透到“银行/国债/回购”层级，写出：周末/节假日赎回能否执行？信息披露的最坏延迟是多少？

### 11.22 俄罗斯 1998（主权违约：外债/财政×政策信誉）
- **Outcome**：主权违约/重组触发全球风险再定价，对手方链条外溢（含 LTCM 等）。  
- **T0 环境**：财政与外债脆弱、商品价格下行、政策信誉不足；资本会“测试防线”。  
- **Killer Assumption**：外部融资与政策承诺能撑过冲击（不会违约）。  
- **Wedge**：资本外流+融资成本上升。  
- **Flywheel**：利差↑→融资更难→财政更差→利差更↑。  
- **First Tripwire**：利差走阔、储备下降、拍卖失败、政策口径反复。  
- **Catastrophic Bound**：融资窗口关闭且政策无法提供可信背书（被迫违约/重组）。  
- **Death Spiral**：资本外流 → 利差飙升 → 融资失败 → 违约/重组 → 更大外流与外溢。  
- **Noise**：短期救助传闻与政治口号。  
- **最短探针（30d）**：把主权风险拆成三条：财政现金流、外币融资能力、政策信誉；任何一条断裂都按违约情景定价。

### 11.23 Argentina 2018（货币危机：利率与资本外逃）
- **Outcome**：在外部融资成本上升时，弱信誉体系更易陷入资本外逃与贬值螺旋。  
- **T0 环境**：经常账户脆弱、财政压力、对外币融资依赖高；外部利率上升会瞬间改写可持续性。  
- **Killer Assumption**：加息足以阻止外逃（忽略信誉与财政约束）。  
- **Wedge**：贬值预期形成后，市场开始“测试防线”。  
- **Flywheel**：贬值预期↑→外逃↑→汇率跌→通胀↑→更强贬值预期。  
- **First Tripwire**：外汇储备下滑、汇率波动扩大、融资拍卖失败。  
- **Catastrophic Bound**：资本管制/IMF 方案仍无法恢复信心（进入长期高通胀/衰退）。  
- **Death Spiral**：贬值→通胀→加息/紧缩→增长更差→信心更差→更贬值。  
- **Noise**：短期“强硬加息”的表演性动作。  
- **最短探针（7d）**：对任何“高外币负债体系”，把偿债来源写成现金流表：外汇收入、储备、再融资；若主要靠再融资，就默认高脆弱。

### 11.24 ERM 1992（黑色星期三：固定汇率被测试）
- **Outcome**：当基本面与政策承诺冲突，固定汇率会被市场攻击，最终放弃防线。  
- **T0 环境**：固定汇率需要储备与政策一致性；当利率/增长约束不允许持续防守，市场会集中攻击。  
- **Killer Assumption**：政策承诺足够可信，且储备/加息能无限防守。  
- **Wedge**：市场预期防线不可持续（基本面与政策目标冲突）。  
- **Flywheel**：攻击→储备消耗/加息→经济更弱→更难防守→更大攻击。  
- **First Tripwire**：储备快速消耗、远期点数异常、政策口径摇摆。  
- **Catastrophic Bound**：防守成本（加息/储备）超过政治与经济可承受上限（被迫退出机制）。  
- **Death Spiral**：攻击 → 防守加息/烧储备 → 经济承压 → 防守意愿下降 → 更大攻击 → 退出。  
- **Noise**：短期“央行表态强硬”。  
- **最短探针（48h）**：对任何“承诺型锚”（汇率/利率/兑付），强制写：它需要哪三种资源支撑？资源上限在哪？一旦被测试，72 小时内你能做什么？

### 11.25 WTI 2020（负油价：仓储约束 × 期货结算机制）
- **Outcome**：当仓储成为硬约束，期货价格可出现极端（负价），保证金与流动性压力外溢。  
- **T0 环境**：供给冲击+需求骤降；现货仓储与交割机制决定“价格能掉到哪里”。  
- **Killer Assumption**：价格下限由“生产成本”决定（忽略仓储与交割约束）。  
- **Wedge**：仓储满 + 交割逼近导致被迫平仓/转仓。  
- **Flywheel**：仓储紧→逼平仓→价格崩→保证金↑→更逼平仓。  
- **First Tripwire**：现货升贴水异常、仓储利用率逼近上限、近月合约流动性变差。  
- **Catastrophic Bound**：交割约束让“必须卖出”的力量超过任何买盘（no-bid）。  
- **Death Spiral**：仓储约束 → 逼平仓/转仓 → 价格崩 → 保证金上调 → 更大平仓。  
- **Noise**：只盯期货曲线，不看仓储与交割规则。  
- **最短探针（7d）**：对任何大宗敞口，强制写“硬约束地图”：仓储/运输/交割/保证金；并假设“规则在极端下会被临时改变”。  

### 11.26 Credit Suisse 2023（AT1 清零/救助并购）
- **Outcome**：信任危机触发被并购救助，AT1 清零，信用层级被改写。  
- **T0 环境**：长期治理与风控失误累积，市场信心脆弱，存款与交易对手高度敏感。  
- **Killer Assumption**：资本工具（AT1）与优先级在极端下不会被“临时改写”。  
- **Wedge**：信心事件 + 负面舆情/社交扩散导致快速外流。  
- **Flywheel**：流动性外流 → 评级与对手方收缩 → 融资成本上升 → 更大外流。  
- **First Tripwire**：存款/prime brokerage 资金快速流出、CDS 飙升、监管“合并/救助”信号。  
- **Catastrophic Bound**：对手方收紧授信 + 资本工具被清零（信用层级重排）。  
- **Death Spiral**：舆情/信心恶化 → 外流 → 资本/流动性承压 → 监管介入 → 被动合并。  
- **Noise**：单日辟谣或管理层“安抚”表态。  
- **最短探针（48h）**：写出“流动性断裂阈值”：存款流出速度、AT1/次级债报价、对手方授信变化；任何一项失控即视为不可自救。  

### 11.27 Thailand 1997（亚洲金融危机：固定汇率崩溃）
- **Outcome**：固定汇率失守，货币大幅贬值，外币负债体系进入违约与衰退。  
- **T0 环境**：短期外债高、储备不足、房地产与信贷泡沫。  
- **Killer Assumption**：通过加息与短期管制即可维持汇率锚。  
- **Wedge**：储备下降 + 市场预期反转。  
- **Flywheel**：贬值预期↑ → 资本外逃↑ → 储备↓ → 更强贬值预期。  
- **First Tripwire**：NDF/远期隐含贬值扩大、短债/储备比恶化、银行外币错配上升。  
- **Catastrophic Bound**：储备低于短期外债覆盖率阈值，被迫放弃汇率锚。  
- **Death Spiral**：攻击 → 贬值 → 负债爆表 → 银行信用收缩 → 经济衰退 → 更大外流。  
- **Noise**：短期“强硬加息”或临时口头干预。  
- **最短探针（7d）**：把外币负债拆成“到期墙 + 真实外汇收入”，若主要依赖再融资则默认高危。  

### 11.28 Iceland 2008（小国大行：银行体系超出主权承载）
- **Outcome**：银行体系崩溃，资本管制与货币贬值，主权救助能力不足。  
- **T0 环境**：银行资产规模数倍于 GDP，依赖批发融资与外币负债。  
- **Killer Assumption**：主权与央行能够无条件兜底。  
- **Wedge**：全球融资冻结 + 货币贬值压力。  
- **Flywheel**：融资断裂 → 资产抛售 → 货币贬值 → 外币负债膨胀 → 更大破产风险。  
- **First Tripwire**：批发融资占比高、FX 掉期价格飙升、对手方缩减额度。  
- **Catastrophic Bound**：银行资产/GDP 规模与外币负债远超央行储备与财政能力。  
- **Death Spiral**：融资断 → 资产抛 → 货币跌 → 资产负债表恶化 → 破产/资本管制。  
- **Noise**：官方担保声明与短期流动性工具。  
- **最短探针（30d）**：测算“银行资产/GDP”与“外币负债/储备”比值；超过阈值则视为主权无力兜底。

## 模式总结（06 版）：金融危机的通用“死法”和最快探针

### A) 反复出现的 Killer Assumption（致命假设 Top 7）
- **融资永远可续作**：短融/回购/滚动融资不会突然断。  
- **相关性不会趋 1**：分散能救命；极端下不会一起跌。  
- **规则不变**：交易所/清算所/监管不会临时改规则（暂停、回滚、闸门）。  
- **“安全资产”永远安全**：危机里仍可卖可融（不出现 no-bid）。  
- **储备可随时兑付**：稳定币/现金等价物在非工作日也能顺畅赎回。  
- **对手方不会同时行动**：不会发生 race to the exit（对手方竞速平仓）。  
- **再融资墙不是硬约束**：到期墙总能被新债覆盖（不会被政策/信心关门）。  

### B) 经典 Death Spiral（把箭头抄出来就能用）
- **保证金螺旋（Margin Spiral）**：波动/利率↑ → 保证金↑ → 被迫卖出 → 价格↓ → 波动更↑ → 保证金更↑  
- **挤兑螺旋（Run Spiral）**：信心↓ → 赎回↑ → 流动性枯竭/闸门 → 更大恐慌 → 更大赎回  
- **短负债长资产（Roll Spiral）**：质疑→短融续不上→抛售→价格崩→更大质疑→更难续作  
- **汇率防线螺旋（Peg Defense Spiral）**：攻击→烧储备/加息→经济更差→防守意愿↓→更大攻击→放弃锚  
- **再融资墙螺旋（Refinancing Wall）**：融资收紧→现金流断→交付/投资断→信心崩→融资更紧  
- **规则风险螺旋（Rule-Risk Spiral）**：极端波动→规则干预→信任下降→流动性更差→更极端波动  

### C) 最快探针（48h/7d/30d，优先级从高到低）
- **48h：现金缺口与抵押品**：在“haircut 上调/波动翻倍/利率跳动”下，你 48h 需要补多少现金/抵押品？从哪里来？  
- **48h：退出可行性**：最坏情景下，你能在几天内以多大滑点退出？（不能退出=高危）  
- **7d：对手方与条款穿透**：对手方数量、集中度、条款触发阈值、race to the exit 情景。  
- **7d：到期墙地图**：未来 3/6/12 个月债务到期+再融资来源（把“会发生”变成表格）。  
- **30d：制度/规则边界**：在极端下谁有权暂停/闸门/回滚？你有没有预案（替代市场/替代通道）？  

### D) 用这份库写 Gate（建议）
- **默认 Gate 2**：只要你还没写出“现金缺口×条款触发×退出可行性”三件套。  
- **不得 Gate 3**：若任何一条成立：规则风险不可控 / 赎回通道不可用 / 对手方不透明 / 无退出路径。
