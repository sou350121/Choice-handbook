## 31.x 材料/化工/基础化学（06 硬化案例）

### 31.1 多源原料体系（相对成功） vs 单一原料依赖（失败模式）
- **Outcome**：多源原料更能抗断供；单一原料依赖在冲击期停产。  
- **T0 环境**：地缘冲突与物流风险上升。  
- **Killer Assumption**：关键原料供应永远稳定。  
- **Wedge**：原料出口限制或事故。  
- **Flywheel**：多源保障↑ → 交付稳定↑ → 现金流稳 → 议价力↑。  
- **First Tripwire**：原料价格异常、单一供应占比上升。  
- **Catastrophic Bound**：关键原料断供超过安全库存周期。  
- **Death Spiral**：断供 → 停产 → 客户流失 → 现金流下降 → 供应更不稳。  
- **Noise**：短期现货价格波动。  
- **最短探针（48h）**：列“Top5 原料依赖度 + 安全库存天数”，低于阈值即判高危。  

### 31.2 严格安全管理（相对成功） vs 事故频发工厂（失败模式）
- **Outcome**：安全体系降低长期停产；事故频发导致监管介入与产能损失。  
- **T0 环境**：化工装置复杂，事故外溢代价高。  
- **Killer Assumption**：小事故不会累积成停产风险。  
- **Wedge**：事故/泄漏引发停产与监管整改。  
- **Flywheel**：安全投入↑ → 事故↓ → 稳定生产↑ → 现金流↑ → 更安全投入。  
- **First Tripwire**：事故频次上升、整改未闭环。  
- **Catastrophic Bound**：重大事故触发长期停产或资质吊销。  
- **Death Spiral**：事故→停产→现金流↓→维护不足→事故更频。  
- **Noise**：短期产量反弹。  
- **最短探针（7d）**：统计“事故率+整改闭环率”，低于阈值即判安全失控。  

### 31.3 有序扩产（相对成功） vs 过度扩产（失败模式）
- **Outcome**：有序扩产维持利润；过度扩产在周期下行时亏损。  
- **T0 环境**：供需周期波动大，建厂周期长。  
- **Killer Assumption**：高景气可持续覆盖新增产能。  
- **Wedge**：需求下滑与价格下行。  
- **Flywheel**：供需平衡 → 价格稳定 → 现金流稳 → 稳健扩产。  
- **First Tripwire**：库存上升、现货价持续下跌。  
- **Catastrophic Bound**：现金成本高于现货价且持续多季。  
- **Death Spiral**：价格↓ → 亏损↑ → 现金流↓ → 维护不足 → 竞争力更弱。  
- **Noise**：单季需求反弹。  
- **最短探针（30d）**：跟踪“库存/销售比 + 现金成本/现货价”，超过阈值即判过度扩产。
