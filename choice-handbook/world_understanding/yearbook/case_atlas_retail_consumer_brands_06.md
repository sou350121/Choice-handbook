## 19.x 零售/消费品牌/渠道（06 硬化案例）

### 19.1 Amazon（成功） vs Sears（失败）
- **Outcome**：Amazon 通过渠道迁移与履约飞轮扩张；Sears 在零售模式转移中衰落。  
- **T0 环境**：电商渗透率上升、消费者对便利性预期提升。  
- **Killer Assumption**：线下客流与百货模式会长期稳定。  
- **Wedge**：履约与价格透明度改变购买决策。  
- **Flywheel**：更多订单 → 履约效率↑ → 价格/体验↑ → 更多订单。  
- **First Tripwire**：同店销售长期下滑、库存周转恶化、线上转化被替代。  
- **Catastrophic Bound**：现金流无法覆盖租约与库存资金占用。  
- **Death Spiral**：客流↓ → 库存积压 → 折扣↑ → 毛利↓ → 关店 → 客流更↓。  
- **Noise**：短期节日促销拉动。  
- **最短探针（7d）**：看“库存周转 + 线上订单占比”，若持续恶化即判渠道迁移失守。  

### 19.2 Costco（成功） vs 传统百货（失速）
- **Outcome**：会员制与规模采购形成护城河；传统高毛利百货在价格透明下失速。  
- **T0 环境**：消费者更重视性价比与信任。  
- **Killer Assumption**：品牌溢价可长期覆盖效率差距。  
- **Wedge**：会员制与精选 SKU 提升周转与议价。  
- **Flywheel**：会员续费↑ → 规模采购↑ → 价格优势↑ → 会员↑。  
- **First Tripwire**：会员续费率下滑、SKU 周转变慢。  
- **Catastrophic Bound**：价格优势消失且会员流失。  
- **Death Spiral**：会员↓ → 规模↓ → 价格劣势 → 会员更↓。  
- **Noise**：短期促销与单季毛利波动。  
- **最短探针（30d）**：追踪“会员续费率 + 单位 SKU 周转”，低于阈值即判飞轮断裂。  

### 19.3 健康 DTC（相对成功） vs 付费流量 DTC（失败模式）
- **Outcome**：有复购与口碑的 DTC 可持续；过度依赖广告获客在成本上升时崩溃。  
- **T0 环境**：广告平台成本上升、隐私限制加强。  
- **Killer Assumption**：CAC 会长期稳定且低于 LTV。  
- **Wedge**：平台隐私规则变化导致投放效率下降。  
- **Flywheel**：复购↑ → 口碑↑ → CAC↓ → 复购↑。  
- **First Tripwire**：CAC 快速上升、复购率下降、退款率升高。  
- **Catastrophic Bound**：LTV/CAC < 1 且无法通过产品/渠道修复。  
- **Death Spiral**：CAC↑ → 亏损↑ → 投放减少 → 规模下降 → 单位成本更高。  
- **Noise**：短期投放拉新与社媒热度。  
- **最短探针（48h）**：计算 LTV/CAC 与复购周期，若连续两期低于阈值即判模式不可持续。
