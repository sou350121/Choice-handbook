## 30.x 医疗服务/照护体系（06 硬化案例）

### 30.1 整合式医疗（相对成功） vs 碎片化收费（失败模式）
- **Outcome**：整合式照护更能控制结果与成本；碎片化体系导致重复检查与成本失控。  
- **T0 环境**：慢病比例上升、医疗成本上行。  
- **Killer Assumption**：按项目收费不会扭曲结果。  
- **Wedge**：激励错配导致过度医疗与重复。  
- **Flywheel**：连续性照护↑ → 结果更好 → 成本更低 → 资源更稳 → 连续性更强。  
- **First Tripwire**：再入院率上升、重复检查比例增加。  
- **Catastrophic Bound**：成本增长超过支付能力，触发强制控费。  
- **Death Spiral**：成本↑ → 资源紧 → 服务质量↓ → 结果更差 → 成本更高。  
- **Noise**：短期门诊量上涨。  
- **最短探针（30d）**：跟踪“再入院率 + 人均成本”，超过阈值即判体系失衡。  

### 30.2 可持续远程医疗（相对成功） vs 补贴驱动远程医疗（失败模式）
- **Outcome**：可持续模式保留高复诊与粘性；补贴驱动在需求退潮后塌陷。  
- **T0 环境**：疫情推动远程医疗普及，补贴与监管宽松。  
- **Killer Assumption**：需求会在补贴与便利下长期维持。  
- **Wedge**：补贴退出与报销政策收紧。  
- **Flywheel**：复诊率↑ → 诊疗效率↑ → 成本更低 → 复诊更高。  
- **First Tripwire**：复诊率下降、医生供给不足。  
- **Catastrophic Bound**：报销政策调整导致单位经济学失效。  
- **Death Spiral**：需求↓ → 医生流失 → 供给下降 → 服务差 → 需求更↓。  
- **Noise**：短期新增用户冲高。  
- **最短探针（7d）**：监测“复诊率 + 医生供给”，低于阈值即判不可持续。  

### 30.3 价值医疗（相对成功） vs 量计费（失败模式）
- **Outcome**：按结果付费推动效率与健康结果；按量收费激励过度诊疗。  
- **T0 环境**：医保与支付方压力上升。  
- **Killer Assumption**：提高服务量即可提升收入并改善结果。  
- **Wedge**：结果考核与成本控制开始落地。  
- **Flywheel**：结果更好 → 支付奖励↑ → 资源投入↑ → 结果更好。  
- **First Tripwire**：质量指标恶化、患者满意度下降。  
- **Catastrophic Bound**：支付方转向结果导向导致原模式收入大幅下滑。  
- **Death Spiral**：质量↓ → 罚款/扣减↑ → 资源下降 → 质量更差。  
- **Noise**：短期手术量增长。  
- **最短探针（48h）**：测“质量指标 + 结果支付占比”，低于阈值即判模式不可持续。
