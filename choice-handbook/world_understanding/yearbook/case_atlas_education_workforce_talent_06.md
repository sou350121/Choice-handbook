## 16.x 教育/技能/人才（06 硬化案例）

### 16.1 Duolingo（成功） vs Rosetta Stone（相对失速）
- **Outcome**：Duolingo 以免费+习惯养成规模化；传统重付费模式增长放缓。  
- **T0 环境**：移动端学习碎片化，用户期望低成本试错。  
- **Killer Assumption**：高价套餐与一次性购买足以长期增长。  
- **Wedge**：免费入口 + 行为激励（streak）提升留存。  
- **Flywheel**：DAU↑ → 学习数据↑ → 内容与节奏更好 → 留存↑ → DAU↑。  
- **First Tripwire**：D7/D30 留存下降、streak 断裂率上升。  
- **Catastrophic Bound**：留存低于阈值，LTV 无法覆盖获客成本。  
- **Death Spiral**：留存差 → 预算削减 → 内容与体验弱 → 留存更差。  
- **Noise**：短期下载量与营销活动。  
- **最短探针（7d）**：监测 D1/D7 留存与 streak 断裂率，低于阈值即判模式不稳。  

### 16.2 Coursera（成功） vs Udacity（转型/收缩）
- **Outcome**：Coursera 以学位/证书合作与企业客户存活；Udacity 从大众 MOOC 转向企业培训。  
- **T0 环境**：证书被用人单位质疑，完成率低。  
- **Killer Assumption**：仅靠课程规模就能形成商业闭环。  
- **Wedge**：与高校/企业合作提升证书认可度。  
- **Flywheel**：合作伙伴↑ → 证书可信↑ → 学员↑ → 伙伴更愿意合作。  
- **First Tripwire**：完成率低、就业兑现下降、合作伙伴减少。  
- **Catastrophic Bound**：证书失去认可，转化与续费长期下滑。  
- **Death Spiral**：完成率低 → 口碑差 → 招生下降 → 伙伴退出 → 更差。  
- **Noise**：短期报名与课程上新数量。  
- **最短探针（30d）**：跟踪完成率与就业转化，低于阈值即判证书不可持续。  

### 16.3 LinkedIn（成功） vs Monster（失速）
- **Outcome**：LinkedIn 通过关系网与身份 SoR 形成平台；传统招聘网站在网络效应上被替代。  
- **T0 环境**：招聘从“职位发布”转向“关系+信任+数据”。  
- **Killer Assumption**：只要有职位列表就能持续吸引求职者。  
- **Wedge**：身份与关系图谱让招聘与内容形成持续活跃。  
- **Flywheel**：更多用户 → 更完整简历与关系 → 招聘方更多 → 职位更多 → 用户更多。  
- **First Tripwire**：活跃用户下降、招聘方续费下滑、职位质量降低。  
- **Catastrophic Bound**：招聘预算向其他平台迁移，网络效应断裂。  
- **Death Spiral**：职位减少 → 求职者减少 → 招聘方更少 → 职位更少。  
- **Noise**：短期职位数量与流量高峰。  
- **最短探针（7d）**：监测活跃简历数与招聘响应率，低于阈值即判网络效应衰退。
