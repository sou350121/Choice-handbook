## 24.x 旅行/酒店/体验经济（06 硬化案例）

### 24.1 酒店直销体系（相对成功） vs 过度依赖 OTA（失败模式）
- **Outcome**：直销与会员体系保持定价权；过度依赖 OTA 让利润被抽走。  
- **T0 环境**：在线分发平台掌握流量入口。  
- **Killer Assumption**：分销抽成可长期承受。  
- **Wedge**：平台抽成提高或算法调整。  
- **Flywheel**：会员直销↑ → 利润↑ → 体验与品牌投入↑ → 会员更↑。  
- **First Tripwire**：OTA 贡献收入占比过高、直销转化停滞。  
- **Catastrophic Bound**：抽成上升导致毛利率跌破盈亏平衡。  
- **Death Spiral**：利润↓ → 体验投入↓ → 直销更弱 → 依赖更强。  
- **Noise**：短期入住率反弹。  
- **最短探针（30d）**：做“渠道利润表”，若 OTA 占比>阈值立即启动直销提升计划。  

### 24.2 航空公司（高固定成本） vs 轻资产出行平台（相对稳健）
- **Outcome**：航空在需求骤降时现金流断裂；轻资产平台更能收缩。  
- **T0 环境**：燃油价格波动、疫情/地缘冲击。  
- **Killer Assumption**：负债与固定成本可以被需求增长长期覆盖。  
- **Wedge**：需求骤降 + 成本刚性。  
- **Flywheel**：需求↑ → 载客率↑ → 单位成本↓ → 盈利↑ → 更高运力。  
- **First Tripwire**：载客率持续低于盈亏线、现金消耗加速。  
- **Catastrophic Bound**：现金流断裂且融资窗口关闭。  
- **Death Spiral**：亏损↑ → 资本成本↑ → 削减航线 → 网络价值↓ → 亏损更↑。  
- **Noise**：短期油价回落。  
- **最短探针（48h）**：测“现金跑道 + 载客率阈值”，低于阈值即判需紧急缩减运力。  

### 24.3 强 IP 主题乐园（相对成功） vs 区域小园（失败模式）
- **Outcome**：强 IP 与多元收入可支撑高资本投入；缺 IP 的小园难以覆盖成本。  
- **T0 环境**：体验消费周期性强、资本开支高。  
- **Killer Assumption**：一次性投资可长期带来稳定客流。  
- **Wedge**：新鲜感衰减导致复购下降。  
- **Flywheel**：IP 吸引力↑ → 客流↑ → 现金流↑ → 更新投入↑ → IP 更强。  
- **First Tripwire**：复游率下降、二次消费减少。  
- **Catastrophic Bound**：现金流不足以支持维护与更新。  
- **Death Spiral**：客流↓ → 维护↓ → 体验差 → 客流更↓。  
- **Noise**：短期节假日客流峰值。  
- **最短探针（7d）**：测“复游率 + 二次消费占比”，低于阈值即判体验飞轮断裂。
