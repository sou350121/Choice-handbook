## 13.x 供应链与制造（06 硬化案例）

### 13.1 Toyota（相对成功） vs “极致 JIT 无冗余体系”（失败）
- **Outcome**：冗余与二供让系统更抗冲击；极致精益在冲击期更脆。
- **T0 环境**：地缘/疫情导致单点断供概率上升。
- **Killer Assumption**（失败者）：供应链永远平稳、交期波动可忽略。
- **Wedge**：关键零部件（芯片/材料）单点。
- **Flywheel**：二供/库存→交付稳定→客户信任→更好议价与计划→更稳定交付。
- **First Tripwire**：关键件交期拉长、供应商财务/地缘风险上升。
- **Catastrophic Bound**：关键件断供导致产线停摆。
- **Death Spiral**：断供 → 停产 → 现金流恶化 → 更难拿到产能 → 断供更严重。
- **Noise**：平时的库存周转 KPI。
- **最短探针（7d）**：做一次“关键件二供审计”：列 Top20 关键件、二供比例、替代周期与库存覆盖天数。

### 13.2 Boeing（失败/受挫）
- **Outcome**：质量事故→监管收紧→交付中断→信任与现金流受损。
- **T0 环境**：复杂供应链 + 成本压力 + 认证严格。
- **Killer Assumption**：把质量与认证当成可被成本/进度挤压的变量。
- **Wedge**：供应链质量与组织治理（认证链条）。
- **Flywheel**（反向）：赶进度→返工/缺陷↑→事故风险↑→监管更严→交付更慢。
- **First Tripwire**：缺陷复发、内部举报、返工率上升。
- **Catastrophic Bound**：重大事故触发停飞/监管冻结。
- **Death Spiral**：事故→监管→停产/交付↓→现金流↓→供应链更脆→事故风险↑。
- **Noise**：订单口径。
- **最短探针（30d）**：看“质量门禁”是否真能 stop-the-line：缺陷率、返工率、供应商不良闭环时长。

### 13.3 Maersk（相对成功/转型） vs Hanjin（失败/破产）
- **Outcome**：资本纪律与资产负债表决定航运周期生死。
- **T0 环境**：航运是强周期、强杠杆行业。
- **Killer Assumption**（失败者）：景气会持续，杠杆扩张不会反噬。
- **Wedge**：运价下行 + 债务到期墙。
- **Flywheel**：资本纪律→更低融资成本→更能熬周期→更强议价。
- **First Tripwire**：运价下行+融资收紧同时出现。
- **Catastrophic Bound**：融资窗口关闭且现金流覆盖不了债务。
- **Death Spiral**：运价↓ → 现金流↓ → 信用↓ → 融资更难 → 运力处置踩踏。
- **Noise**：单边景气期的利润。
- **最短探针（48h）**：把“到期墙”画出来：未来 12 个月债务与现金流覆盖倍数。

### 13.4 Zara（成功） vs Gap（相对失速）
- **Outcome**：快反供应链更能匹配需求波动；慢周期库存更易爆。
- **T0 环境**：需求变化更快，库存风险上升。
- **Killer Assumption**（失败者）：预测足够准确，可以提前大量备货。
- **Wedge**：补货周期与近岸产能。
- **Flywheel**：更快补货→更少库存折扣→更高现金流→更能投资快反。
- **First Tripwire**：折扣依赖上升、库存周转恶化。
- **Catastrophic Bound**：库存积压导致现金流断裂。
- **Death Spiral**：库存积压→折扣→毛利↓→现金流↓→更难调整供应链→库存更积压。
- **Noise**：单季爆款。
- **最短探针（7d）**：统计补货周期（从信号到上架），以及折扣销售占比。

### 13.5 Apple（成功） vs “只做品牌、制造不可控”（失败模式）
- **Outcome**：用资本与治理换交付确定性；制造不可控者在冲击期断供。
- **T0 环境**：关键元件产能稀缺，交付决定收入。
- **Killer Assumption**（失败者）：代工会自然优先你。
- **Wedge**：关键元件锁定（预付款/长约/多源）。
- **Flywheel**：交付稳定→销量稳定→现金流强→更能锁产能→交付更稳。
- **First Tripwire**：关键元件单点、良率/质量波动上升。
- **Catastrophic Bound**：关键元件断供导致旗舰产品延期。
- **Death Spiral**：延期→渠道信任↓→需求波动↑→供应链计划更难→延期更频繁。
- **Noise**：短期毛利波动。
- **最短探针（30d）**：列 Top10 关键元件：二供比例、库存覆盖天数、替代周期、合同约束。

### 13.6 TSMC（成功） vs “无产能保障的 Fabless”（失败模式）
- **Outcome**：产能与良率执行决定交付；无保障者在挤兑期被卡死。
- **T0 环境**：先进制程产能稀缺，地缘风险上升。
- **Killer Assumption**（失败者）：需要时总能买到产能。
- **Wedge**：产能锁定与工艺路线选择。
- **Flywheel**：良率与信任→更多客户→更多资本开支→更高良率。
- **First Tripwire**：交期拉长、客户被迫改版或转单。
- **Catastrophic Bound**：关键节点被断供/出口管制。
- **Death Spiral**：交付不稳→客户流失→规模下降→良率与投资下降→交付更不稳。
- **Noise**：单代性能。
- **最短探针（7d）**：看产能合同（有无 long-term agreement）、多代工备选与可移植性。

### 13.7 “极致降本外包”（失败） vs “关键环节内生能力”（成功）
- **Outcome**：外包关键能力在冲击期暴露脆弱；内生或强绑定提高生存力。
- **T0 环境**：冲击期合同与价格机制失灵。
- **Killer Assumption**（失败者）：合同可以替代控制权。
- **Wedge**：关键工序/关键质量门禁。
- **Flywheel**：控制关键环节→质量稳定→交付稳定→客户信任→更强议价。
- **First Tripwire**：外包方产能/质量波动、交付延期变常态。
- **Catastrophic Bound**：关键外包方倒闭/被限制/断供。
- **Death Spiral**：延期→返工↑→成本↑→现金流↓→更依赖外包→更延期。
- **Noise**：短期毛利好看。
- **最短探针（48h）**：做“关键路径地图”：哪些环节不能外包？哪些必须有二供？

### 13.8 Samsung Note7（失败：质量事故） vs “可停线的质量门禁体系”（成功）
- **Outcome**：电池缺陷与召回导致品牌与现金流受损；能真正 stop-the-line 的质量体系更可持续。  
- **T0 环境**：高节奏迭代、供应链复杂，质量风险在规模化下指数放大。  
- **Killer Assumption**：可以用速度与营销覆盖质量风险。  
- **Wedge**：电池/关键件的质量与一致性（供应商与设计耦合）。  
- **Flywheel**（反向）：赶进度→抽检不足→缺陷漏出→事故→召回→更赶进度。  
- **First Tripwire**：不良率/客诉上升、返工率上升、供应商变更频繁。  
- **Catastrophic Bound**：出现安全事故触发监管/航空禁运/大规模召回。  
- **Death Spiral**：事故→召回→现金流压力→供应链更紧→质量更难控→再事故。  
- **Noise**：发布会声量、单次评测。  
- **最短探针（7d）**：把关键件（电池/结构件）质量指标与闭环时长拉出来：缺陷从发现到彻底根因关闭需要多久？

### 13.9 口罩/PPE（2020）：有冗余与本地化产能（成功） vs 单一低成本全球采购（失败）
- **Outcome**：冲击期“最低成本”体系断供；有冗余与可切换产能者保交付。  
- **T0 环境**：需求暴增+物流受限+出口限制，价格机制失灵。  
- **Killer Assumption**：全球供应永远可买到，且价格机制会自动调节。  
- **Wedge**：本地/近岸产能与原材料替代路线。  
- **Flywheel**：有交付→有现金流→能抢到原料/产能→更有交付。  
- **First Tripwire**：交期突然拉长、出口限制信号、关键原料价格飙升。  
- **Catastrophic Bound**：关键物资断供导致业务/医疗系统停摆（不可逆损失）。  
- **Death Spiral**：断供→抢购→价格失真→更断供→系统性停摆。  
- **Noise**：平时的采购降本 KPI。  
- **最短探针（48h）**：列“关键物资清单”并做二供/替代审计：每个物资给出本地替代周期与最低库存天数。

### 13.10 Takata 气囊（失败：缺陷扩散 × 召回级联）
- **Outcome**：单一关键部件缺陷在规模化下扩散为全球召回与诉讼，供应链与品牌现金流被长期拖累。  
- **T0 环境**：汽车安全件高度认证、替换成本极高；缺陷一旦进入装车会形成“时间延迟的爆雷”。  
- **Killer Assumption**：缺陷能被局部化、不会跨型号/跨客户扩散。  
- **Wedge**：关键材料/工艺缺陷 + 质量门禁缺位。  
- **Flywheel**（反向）：更多装车→更多事故样本→召回扩大→现金流压力→质量更难控。  
- **First Tripwire**：客诉与事故信号上升、同类缺陷复发、供应商质量闭环时间变长。  
- **Catastrophic Bound**：被监管/媒体确认存在系统性安全风险（不可逆召回）。  
- **Death Spiral**：缺陷漏出 → 事故/诉讼 → 召回扩大 → 现金流/产能紧 → 质量更差 → 更大召回。  
- **Noise**：短期成本/交付 KPI；“概率很低”的自我安慰。  
- **最短探针（30d）**：做一次“缺陷传播半径”审计：同工艺/同材料/同批次覆盖了多少车型与客户？一旦召回，替换产能与现金流上限是多少？

### 13.11 婴儿配方奶粉（2022：单厂停摆 → 全国性短缺）
- **Outcome**：单一关键工厂的质量/合规事件即可导致系统性短缺，补产与认证周期长。  
- **T0 环境**：食品/医药类产能高度合规，切换工厂与扩产速度受硬约束。  
- **Killer Assumption**：关键品类的供给可被快速替代（换厂/换证很快）。  
- **Wedge**：合规/污染事件触发停产与召回。  
- **Flywheel**（反向）：停产→短缺→恐慌抢购→渠道失真→更短缺。  
- **First Tripwire**：质量审计缺陷、关键批次异常、监管关注度上升。  
- **Catastrophic Bound**：停产+召回持续时间超过库存覆盖天数（进入社会事件）。  
- **Death Spiral**：停产 → 供给断裂 → 抢购 → 渠道失衡 → 更断裂。  
- **Noise**：平时的“供应充足”惯性。  
- **最短探针（7d）**：对任何关键品类做“单厂依赖度”表：最大供给方占比、替代认证周期、最低安全库存天数。

### 13.12 汽车芯片短缺（2021：关键件单点 × JIT 脆弱性）
- **Outcome**：当关键芯片成单点，整车交付被卡死；库存与合同条款决定谁先恢复。  
- **T0 环境**：半导体长周期、产能不可瞬间扩；汽车链路长且认证严格。  
- **Killer Assumption**：关键件可以像普通零件一样随时补货（交期短、可替换）。  
- **Wedge**：关键 MCU/功率器件/传感器供给收缩。  
- **Flywheel**（反向）：缺芯→停产→现金流下降→更难锁产能→更缺芯。  
- **First Tripwire**：交期拉长、供应商配额化、同类行业开始抢产能。  
- **Catastrophic Bound**：停线持续超过现金流承受期（订单流失/渠道崩）。  
- **Death Spiral**：缺芯 → 停产 → 交付延期 → 客户/渠道转向 → 规模下降 → 更难拿到产能。  
- **Noise**：只看总库存不看“关键件库存”。  
- **最短探针（48h）**：列 Top20 关键件（按“停线影响”排序），写出二供/替代方案与最短切换周期；没有二供的就是红线。

### 13.13 苏伊士运河（2021：物流通道单点）
- **Outcome**：单一咽喉通道阻塞能把全球交期拉长并放大库存/现金流压力。  
- **T0 环境**：全球化供应链把运输通道当“免费且稳定”，忽略事故与地缘风险。  
- **Killer Assumption**：物流通道不构成硬约束（延迟可忽略）。  
- **Wedge**：通道中断导致在途库存冻结。  
- **Flywheel**：延迟→缺货→抢运/改道→成本↑→更难准时→更大延迟。  
- **First Tripwire**：航运拥堵、运价/保险费上升、替代路线容量不足。  
- **Catastrophic Bound**：关键零部件在途冻结超过停线阈值。  
- **Death Spiral**：通道中断 → 在途冻结 → 停线/缺货 → 抢运改道 → 成本与延迟更大。  
- **Noise**：把它当成“短期小事故”而不做路线冗余。  
- **最短探针（7d）**：画运输依赖图：关键件走哪条通道？替代路线容量与成本？在途库存冻结 14 天会怎样？

### 13.14 Pratt & Whitney GTF（2023：缺陷扩散 × 产能修复硬约束）
- **Outcome**：关键部件缺陷触发大规模检查/停飞，影响航司运力与制造商交付节奏。  
- **T0 环境**：航空发动机高度认证与长周期维护；修复产能与备件是硬约束。  
- **Killer Assumption**：缺陷能被快速定位且修复产能足够。  
- **Wedge**：关键材料/工艺问题导致强制检查与拆解。  
- **Flywheel**（反向）：停飞↑→备件需求↑→修复排队↑→停飞更久。  
- **First Tripwire**：同类缺陷在不同批次复发、维护周期被拉长、备件交期上升。  
- **Catastrophic Bound**：修复产能不足导致运力缺口长期化（业务模型被改写）。  
- **Death Spiral**：缺陷暴露 → 强制检查 → 备件/修复瓶颈 → 停飞延长 → 更大瓶颈。  
- **Noise**：把它当成“单一产品事故”而忽略“修复产能=硬约束”。  
- **最短探针（30d）**：对任何关键系统做“修复产能审计”：若缺陷率上升，你的检测/返修/备件链条最大吞吐是多少？超过就进入长期断供。

### 13.15 合同制造：Emergent/J&J（失败：代工质量治理）
- **Outcome**：代工厂质量失控导致批次报废与交付延期，信任与监管成本上升。  
- **T0 环境**：应急需求下追速度，代工链条复杂；一旦污染/混线，损失不可逆。  
- **Killer Assumption**：合同条款与审计可以替代质量文化与现场门禁。  
- **Wedge**：混线/污染/记录不全等质量事故。  
- **Flywheel**（反向）：赶进度→降低门禁→事故→停产整顿→更赶进度。  
- **First Tripwire**：偏差报告增加、审计发现项复发、批次合格率下降。  
- **Catastrophic Bound**：监管介入导致长期停产/撤销资格。  
- **Death Spiral**：事故→停产→交付压力→更冒险→更大事故→更长停产。  
- **Noise**：短期产量数字与“按时交付”的口号。  
- **最短探针（7d）**：把“质量门禁”写成硬阈值：任何关键偏差必须 stop-the-line；并核对可追溯性（批次/记录/人员/设备）是否完整。

### 13.16 认证材料/锻造单点（失败模式：替代周期极长）
- **Outcome**：当关键材料/锻造/认证供应高度集中，地缘/制裁/事故会把替代周期拉到“以年计”。  
- **T0 环境**：航空/军工等行业的材料与工艺认证链条长，切换供应商是工程项目，不是采购动作。  
- **Killer Assumption**：关键原材料/锻造产能可在短期内替代。  
- **Wedge**：单一供应商受限（地缘/事故/制裁/能源）。  
- **Flywheel**（反向）：供给受限→交付延期→现金流下降→投资下降→供给更受限。  
- **First Tripwire**：供应集中度上升、认证路径缺失、替代工艺没有预案。  
- **Catastrophic Bound**：单点供应断裂超过认证/切换周期（交付长期归零）。  
- **Death Spiral**：断供 → 延期 → 现金流恶化 → 更难投资替代 → 更长期断供。  
- **Noise**：短期低价与“供应稳定”的惯性。  
- **最短探针（30d）**：对关键材料/工艺做“认证替代清单”：替代供应商是谁、认证要多久、最坏情景下库存能撑几个月。  

### 13.17 富士康郑州 2022（单一园区/劳动力冲击）
- **Outcome**：核心生产园区受疫情/劳动力冲击导致出货大幅下滑。  
- **T0 环境**：产能高度集中、用工密集、交付节奏紧。  
- **Killer Assumption**：单一超大园区在任何冲击下都能维持稳定运转。  
- **Wedge**：封控/劳动力流失/通勤受阻。  
- **Flywheel**：停工 → 交付延迟 → 赶工加班 → 质量与稳定性下降 → 更大停工。  
- **First Tripwire**：当地公共事件升级、人员流动受限、临时工比例激增。  
- **Catastrophic Bound**：园区停摆超过数周且无可替代产能。  
- **Death Spiral**：停摆 → 订单积压 → 赶工 → 质量事故 → 复工更慢 → 更大积压。  
- **Noise**：短期“增产承诺”与调班公告。  
- **最短探针（7d）**：列出 Top1/Top2 产能园区占比与替代周期；任一园区>50%即强制设 Gate 2。  

### 13.18 Renesas 2021（工厂火灾：关键产线停摆）
- **Outcome**：关键芯片产线因火灾停产，恢复周期拉长，引发产业链缺口。  
- **T0 环境**：单厂/单线承担关键型号供给，库存低。  
- **Killer Assumption**：单厂事故对整体交付影响可控。  
- **Wedge**：事故/停电导致产线停摆与设备损伤。  
- **Flywheel**：停产 → 交付延期 → 现金流承压 → 修复投入不足 → 更慢恢复。  
- **First Tripwire**：维护延期、设备老化警报、产线事故频发。  
- **Catastrophic Bound**：恢复周期超过 3–6 个月且无替代产能。  
- **Death Spiral**：停产 → 缺货 → OEM 减产 → 收入下降 → 复产更慢 → 更长缺货。  
- **Noise**：短期“复工时间表”反复修正。  
- **最短探针（30d）**：做“关键工厂恢复时间表 + 替代产能地图”，超过阈值即视为长期缺口。  

### 13.19 德州寒潮 2021（能源/基础设施中断）
- **Outcome**：电力与水供应中断导致晶圆厂/化工链条停产，恢复期拉长。  
- **T0 环境**：生产高度依赖稳定的电力/水/化学品供应。  
- **Killer Assumption**：公共基础设施在极端天气下仍可用。  
- **Wedge**：电网故障/供水中断引发停机与污染风险。  
- **Flywheel**：停机 → 设备清洗/复产耗时 → 交付延期 → 现金流承压 → 维护更难。  
- **First Tripwire**：电网预警升级、关键原料库存下滑、厂区负荷限制。  
- **Catastrophic Bound**：持续停电/停水导致产线污染或设备损伤。  
- **Death Spiral**：停机 → 复产慢 → 供给缺口 → 订单调整 → 现金流下降 → 维护更难。  
- **Noise**：短期气象好转与“复工公告”。  
- **最短探针（7d）**：列出关键公共资源依赖度（电/水/气/化学品）与应急备份时长。

## 模式总结（06 版）：供应链与制造的通用“死法”和最快探针

### A) 反复出现的 Killer Assumption（致命假设 Top 7）
- **关键件可替代**：把“认证/工装/良率爬坡”当成采购动作而不是工程项目。  
- **交付延迟可承受**：低估停线阈值与现金流断裂速度。  
- **质量可以用返工解决**：忽略缺陷扩散（召回/停产/监管）是不可逆的。  
- **合同=控制权**：外包关键路径后仍以为条款能替代质量门禁与现场治理。  
- **通道不会被卡住**：把物流咽喉/地缘当成低概率小事。  
- **最低成本是最优**：把降本 KPI 当目标，牺牲冗余与退出权。  
- **修复产能无限**：忽略检测/返修/备件链条是硬约束（会排队、会饱和）。  

### B) 经典 Death Spiral（最常用的箭头链条）
- **单点断供螺旋**：关键件断供 → 停线/交付延期 → 现金流恶化 → 更难锁产能/备货 → 更断供  
- **质量事故螺旋**：赶进度 → 门禁下降 → 缺陷漏出 → 召回/停产/监管 → 现金流压力 → 更赶进度  
- **库存杠杆螺旋**：预测失准 → 库存积压 → 折扣/毛利下滑 → 现金流变差 → 更难调整供应链 → 更积压  
- **通道咽喉螺旋**：通道中断 → 在途冻结 → 缺货/停线 → 抢运改道 → 成本与延迟更大 → 更难准时  
- **修复产能螺旋**：缺陷暴露 → 强制检查/返修 → 产能瓶颈 → 排队延长 → 供给长期化缺口  

### C) 最快探针（48h/7d/30d）
- **48h：关键路径地图**：Top20 关键件（按“停线影响”排序）+ 二供/替代周期 + 当前库存覆盖天数。  
- **7d：单点/通道审计**：单厂/单证书/单通道依赖度；替代路线容量与成本。  
- **7d：质量门禁阈值**：缺陷率/返工率/闭环时长（stop-the-line 的触发条件写死）。  
- **30d：认证与工装替代清单**：替代供应商是谁？认证要多久？工装/配方/工艺迁移的最坏耗时是多少？  
- **30d：修复产能审计**：检测/返修/备件最大吞吐；超过即进入“长期缺口”。  

### D) 用这份库写 Gate（建议）
- **默认 Gate 2**：只要你还没写出“关键件 Top20 + 二供/替代周期 + 停线阈值”。  
- **不得 Gate 3**：若任一条成立：关键件无二供且替代>3–6 个月（示意）/ 质量门禁不可 stop-the-line / 修复产能明显不足。
