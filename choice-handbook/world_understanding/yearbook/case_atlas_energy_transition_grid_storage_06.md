## 22.x 能源转型/电网/储能（06 硬化案例）

### 22.1 高渗透风光（相对成功） vs 电网滞后（失败模式）
- **Outcome**：电网升级与调度能力提升能承接高渗透可再生；电网滞后导致频繁限电与事故。  
- **T0 环境**：风光装机快速增长、负荷波动加剧。  
- **Killer Assumption**：装机增长自然带来稳定供电。  
- **Wedge**：电网调度与输配能力不足。  
- **Flywheel**：电网升级 → 消纳能力↑ → 投资效率↑ → 电网更升级。  
- **First Tripwire**：弃风弃光率升高、频率波动与事故增加。  
- **Catastrophic Bound**：大范围停电或系统性可靠性事件。  
- **Death Spiral**：事故↑ → 投资放缓 → 可靠性更差 → 事故更多。  
- **Noise**：短期装机规模排名。  
- **最短探针（30d）**：测“弃风弃光率 + 电网事故率”，超过阈值即判系统不稳。  

### 22.2 储能+需求响应（相对成功） vs 纯发电扩张（失败模式）
- **Outcome**：储能与需求响应提升削峰能力；仅靠扩大发电导致成本高且调度僵硬。  
- **T0 环境**：负荷峰谷差加大，燃料价格波动。  
- **Killer Assumption**：只要增加装机就能解决可靠性问题。  
- **Wedge**：峰值负荷与供应错配。  
- **Flywheel**：储能规模↑ → 削峰能力↑ → 可靠性↑ → 投资回报↑。  
- **First Tripwire**：尖峰电价频繁、备用容量不足。  
- **Catastrophic Bound**：尖峰时段供电不足导致强制限电。  
- **Death Spiral**：限电 → 信任下降 → 投资受阻 → 供给更紧 → 限电更多。  
- **Noise**：短期燃料价格回落。  
- **最短探针（7d）**：统计“尖峰缺口 + 储能覆盖小时数”，低于阈值即判削峰失败。  

### 22.3 新建核电（超支/延期） vs 现有核电延寿（相对稳健）
- **Outcome**：新建核电项目易超支与延期；延寿改造更具成本可控性。  
- **T0 环境**：监管严格、供应链复杂、资本成本高。  
- **Killer Assumption**：大型基建可以按期按预算交付。  
- **Wedge**：审批/安全要求变化与供应链延误。  
- **Flywheel**：延寿效率↑ → 可靠电源↑ → 收入稳定↑ → 延寿投入↑。  
- **First Tripwire**：工期延误、成本超支率上升。  
- **Catastrophic Bound**：资本成本上升导致项目财务不可行。  
- **Death Spiral**：延期 → 成本↑ → 融资更难 → 再延期 → 取消。  
- **Noise**：短期政策口号与补贴承诺。  
- **最短探针（30d）**：跟踪“工期偏差 + 资本成本”，超阈值即判新建不可持续。
