# 2026-02-03 | Event Reasoning mini-06 | 小米：自动股份购回（HK$25亿）与林斌减持计划（US$20亿）引发的“口径/信号”争议

## Event Summary

- “港元回购/美元减持”的争议本质不是币种，而是市场在问：公司在用回购传递信心与股东回报， 同时核心高管通过减持计划抽取流动性去做投资基金，这两条信号叠加会不会形成持续的“供给过hang + 估值折价”。 投资上应把注意力从叙事转回可观测变量：回购的真实执行节奏/注销机制、减持计划的启动时点与实际卖出路径、 以及随后的业绩与指引是否支持“回购=低估”的隐含假设。

2) disposal_plan_not_executed_or_slow

## Tripwire / Bound / Spiral

- First Tripwire: 未来 10 个交易日的回购披露显示执行显著弱于口径（或计划提前终止）/出现高管实际卖出且节奏超预期
- Catastrophic Bound: 出现集中/不透明的实际减持（或多名高管/大股东同步减持）并叠加基本面负面（利润/指引下修），导致“回购托底”逻辑失效
- Death Spiral: 口径争议 -> 情绪恶化 -> 卖压+过hang -> 估值下修 -> 回购被动托底/弹药不足 -> 再次恶化

## 最短探针

- 48h: 把两条公告补齐到一手 PDF（尤其回购计划），并拉取最近 10 个交易日的 Next Day Disclosure Return 统计回购节奏
- 7d: 建立“净回购 vs 预期减持上限”的时间序列对照（按周更新），并标注任何实际权益披露变化
- 30d: 以财报窗口为锚点复盘：回购是否穿越限制期、现金/投资变化是否支持持续回购、市场是否重新定价治理信号
