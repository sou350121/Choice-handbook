## 01 结论（人话 3 句）

- **发生了什么**：道通科技：美国“国安/出口管制”链条外溢到充电桩业务的风险
- **核心判断**：“道通科技”最核心的不确定性不是行业景气，而是美国把“国安/数据/关键基础设施”框架外溢到充电桩后， 其数字能源（充电桩）业务在北美的准入/采购/补贴资格可能被行政动作重新定价； 这类事件的投资关键不是新闻热度，而是有没有出现可执行的“制度性动作”（例如：正式调查、采购禁令、NEVI 资格变化、实体清单/1260H 外溢到充电桩主体）。

- **行动含义**：Gate=Gate 2；优先做“跟踪 + 条件动作”，并提前写好停机点（Tripwire/Bound）

## 02 机制与市场传导（你该盯什么变量）

**传导链（简版）**
- 国会/行政层国安叙事 -> 监管/采购口径升级 -> 北美项目方/运营商风险偏好下降 -> 订单延迟/替换
- “Entity List/1260H”对同集团主体的既有定性 -> 充电桩主体被连坐审查 -> 合规成本上升/销售周期变长
- 关键基础设施（电网连接）+ 数据收集叙事 -> 州/联邦层采购禁令或补贴资格约束 -> TAM 突降
- 地缘政治 -> 供应链/芯片/软件依赖与合规审查 -> 毛利/交付/认证周期波动

**潜在受益方**
- 北美/欧盟本土或“可信供应链”充电桩与能源管理厂商（替代受益）
- 合规/安全审计/供应链溯源相关服务（企业合规支出上升的受益方）

**潜在受损方**
- 道通科技的北美数字能源业务（若准入/采购/补贴资格被收紧）
- 任何被贴上“国安不确定性”标签、依赖政府/公用事业采购的海外设备商

**观察名单（最小集合）**
- BIS Entity List / DoD 1260H 名单是否出现“充电桩主体/相关实体”新增或解释口径外扩
- NEVI / FHWA / 州 DOT 的“合规与供应链”采购条款是否针对性收紧
- 关键客户/渠道是否出现延期、替换、暂停招标等信号（合同层面比口头更重要）
- 公司对北美本地化生产、BABA/NEVI 资质、数据治理的披露与第三方验证

## 03 情景推演（带概率）

**1) Base_case_停留在政治信函_但商业摩擦上升**
- 概率: 65% | 时间窗: 3-12_months
- 发生条件: 无实体清单/禁令升级；但采购与法务审查变严
- 事件展开: 北美客户增加尽调与合规条款，销售周期变长、部分订单延迟；公司通过本地化与合规投入对冲
- 市场影响: 增长预期小幅下修、估值折价扩大；但不至于“断崖式”出清

**2) Downside_case_制度性升级_采购禁令或补贴资格受限**
- 概率: 20% | 时间窗: 1-9_months
- 发生条件: 商务部/国防部或州/联邦采购口径转为“限制/禁用”；NEVI 相关资格出现负面变化；或名单外溢
- 事件展开: 关键客户暂停采购/替换供应商，渠道收缩，北美增长断档并外溢到欧洲大客户的风险审查
- 市场影响: 数字能源增长曲线被重估，盈利与现金流预期下修，估值显著压缩

**3) Upside_case_合规与本地化证明有效_风险淡化且欧洲增长对冲**
- 概率: 15% | 时间窗: 6-18_months
- 发生条件: 没有进一步制度动作；公司提供可信第三方验证（供应链/数据/合规），且欧洲收入占比抬升
- 事件展开: “国安叙事”没有形成可执行禁令，市场把事件降级为噪声；公司在欧洲放量对冲北美波动
- 市场影响: 估值折价回撤，市场重新聚焦基本面与海外扩张

## 04 行动方案（默认动作 + 条件动作）

- **默认动作**: 以 Gate 2 处理：不做“拍脑袋方向性重仓”，建立信号板（名单/NEVI/采购口径/客户订单）并做情景化仓位
- **触发 Tripwire 后**: 降低暴露或用对冲保护；把关注点从“增长叙事”切换到“生存/准入”——优先验证订单与采购条款
- **触发 Bound 后（止损/撤退）**: 视为 Gate 3（负面确认）：按制度性断档处理（撤退/止损/重估），并将事件写入长期风险档案
- **对冲/保险**:
  - Pair trade（概念）：受益的本土/可信供应链充电桩厂商 vs 受压的高不确定性海外设备商
  - 用期权/仓位上限管理尾部风险（如果你有衍生品工具）

## 05 风险与停机点（这部分决定你会不会亏大钱）

**Killer assumptions（被打脸就要退）**
- 数字能源在公司增量里已足够“重要”，一旦北美准入收缩会实质影响增长轨迹/估值
- 美国监管不会仅止于政治信函，而会出现可执行的制度动作（采购口径/合规门槛/名单）
- 客户侧对“合规不确定性”反应是延迟/替换（而非继续采购并忽略风险）

- **Tripwire（第一次警报）**: 商务部/国防部启动正式程序（公开函件/调查/立项）或州/联邦采购/NEVI 条款出现针对性排除
- **Bound（硬红线）**: 充电桩主体被加入 Entity List/1260H 或出现联邦/州层面的明确采购禁令/NEVI 不合格结论
- **Spiral（负反馈螺旋）**: 国安叙事 -> 采购冻结 -> 渠道收缩 -> 收入下滑 -> 更难证明合规与本地化 -> 进一步审查/更大折价

- 48h: 把“证据链”补齐到一手来源：国会信函 + BIS FR（Entity List）+ DoD FR（1260H）+ 公司公告（收入结构）
