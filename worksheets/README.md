# Worksheets（练习表单）

这些表单的目标是：把“想清楚”变成“写清楚”，把“感觉”变成“可复盘的证据链”。

## 表单清单（MVP）
- `decision_one_pager.md`：一页决策卡（最常用）
- `premortem.md`：事前验尸（失败尽头）
- `postmortem.md`：事后复盘（真实尽头）

## 使用建议
- 先用 `decision_one_pager.md` 选方向/选项目/选合作。
- 再用 `premortem.md` 找出最可能的失败路径，并写清预防与止损。
- 执行一段时间后用 `postmortem.md` 复盘：尽头在哪里、为什么停、下次如何更早看见。
