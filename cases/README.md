# Cases（案例库）

案例的目的不是给你“标准答案”，而是训练你：
- 在真实约束下做选择（钱/时间/精力/关系/能力/制度）
- 写出“尽头”（停止之处）与退出条件
- 把大选择拆成小实验

## MVP 案例（跨场景）
1. `01_career_switch_or_stay.md`：职业——跳槽还是留下
2. `02_partner_or_alone.md`：关系——继续投入还是止损
3. `03_health_rebuild.md`：健康——从透支到重建系统
4. `04_money_invest_or_cashflow.md`：金钱——投资机会 vs 现金流安全
5. `05_project_choose_and_kill.md`：做事——选项目与“杀项目”
